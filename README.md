# Social-Networking

- Docker
- Docker Compose
- Git
- Postman (for API testing)

### Clone the Repository
git clone https://github.com/vigneshs-1932/Social-Networking

### Go to Project Directory
cd Social-Networking

### Build the Project
docker-compose up

### Stop the Project
docker-compose down
